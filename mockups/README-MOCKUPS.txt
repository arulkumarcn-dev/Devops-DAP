HOW TO CREATE PNG/JPG IMAGES FROM HTML MOCKUPS:

1. Open each HTML file in your browser (Chrome, Edge, Firefox)
   - Right-click on the HTML file
   - Select "Open with" > Your preferred browser

2. Take screenshots using Windows Snipping Tool:
   - Press: Windows Key + Shift + S
   - Select the area to capture
   - The screenshot is copied to clipboard
   - Open Paint or any image editor
   - Paste (Ctrl+V) and save as PNG or JPG

3. Alternative method - Full page screenshot:
   - Open HTML in browser
   - Press F12 (Developer Tools)
   - Press Ctrl+Shift+P
   - Type "screenshot"
   - Select "Capture full size screenshot"
   - Image will be saved automatically
